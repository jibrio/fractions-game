# Changelog

All notable changes to Cool Fractions, newest first.

## 2026-06-15 — endless mode
- The start screen now offers two ways to play: **Learn** (the 20 curated
  levels) and **Endless** (an ever-harder challenge).
- Endless mode procedurally generates levels whose difficulty climbs with the
  stage number — bigger denominators, non-unit fractions, more answer options,
  and multi-step puzzles (order, make-a-whole) once it warms up.
- Endless shows a **streak + best** counter; a wrong answer resets the streak.

## 2026-06-15 — first build
- 20 levels spanning halves → fifths, eighths, equivalence, ordering and
  "make a whole", with the puzzle type changing level to level and difficulty
  ramping toward the end.
- 9 puzzle mechanics: slice, break-the-bar, shade, fill, number line, build a
  whole, pick the match, order smallest→biggest, find two that make a whole.
- Diverse SVG cartoon characters; dark neon visual style.
- Synthesised audio (sound effects + level-complete celebration) with a mute
  toggle. Correct answers shuffle each load.
- Personalised praise banner shown above the characters on each win.
